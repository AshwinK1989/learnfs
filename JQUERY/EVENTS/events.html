Selecting with JQuery

You can use $ sign to select elements from the dom. It will feel a lot like selecting using querySelectorAll()

For example:

  • Selecting all a tags
    • $('a')
  • Selecting all elements from the class .container
    • $('.container')
  • Selecting all elements with id special
    • $('#special')
